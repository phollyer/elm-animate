# 3D Animations

Elm Animate supports 3D transforms for Rotate, Scale, and Translate properties. 3D animations are GPU-accelerated and enable rich, immersive effects.

## Setting Up 3D

3D animations require perspective to create depth. Without perspective, they will appear flat.

Perspective is set using the `Anim.View3D` module, which provides type-safe CSS attributes for 3D setup.

```elm
import Anim.Extra.View3D as View3D exposing (PerspectiveOrigin(..))

view model =
    div 
        [ id "container"
        , View3D.perspective 1000
        , View3D.perspectiveOrigin Center
        ]
        [ div 
            ([ id "card" 
             , View3D.backfaceVisibility View3D.Hidden
             ] ++ CSS.transitionAttributes "card" model.animation)
            [ text "I flip in 3D!" ]
        ]
```

!!! note "Container Selection"
    - For typical UI animations (card flips, hover effects, etc.), you almost always want per-container perspective. A card in the corner of the page should flip the same way as one in the center.
    - For immersive 3D scenes where you want depth across the whole page, root-level perspective makes sense.

## View3D Module

The `Anim.View3D` module provides four key functions:

### perspective

Set the perspective depth on a container element:

```elm
View3D.perspective 1000  -- 1000px perspective depth
```

### perspectiveOrigin

Control where the viewer is looking from:

```elm
View3D.perspectiveOrigin TopLeft      -- Look from top-left
View3D.perspectiveOrigin Center       -- Look from center (default)
View3D.perspectiveOrigin (Percent 25 75)  -- Custom position
View3D.perspectiveOrigin (Px 100 50)      -- Pixel position
```

Available positions: `Center`, `TopLeft`, `TopCenter`, `TopRight`, `LeftMiddle`, `RightMiddle`, `BottomLeft`, `BottomCenter`, `BottomRight`, `Percent Float Float`, `Px Float Float`

### backfaceVisibility

Control visibility of element back faces:

```elm
View3D.backfaceVisibility View3D.Hidden   -- Hide when facing away
View3D.backfaceVisibility View3D.Visible  -- Always visible (default)
```

### transformStyle

Enable nested 3D context:

```elm
View3D.transformStyle View3D.Preserve3D  -- Enable nested 3D
View3D.transformStyle View3D.Flat        -- Flatten children (default)
```

### Perspective Values

| Value | Effect |
| ------- | -------- |
| 500-800px | Dramatic, close-up 3D effect |
| 1000-1500px | Natural, balanced perspective |
| 2000px+ | Subtle, distant 3D effect |

## Translate in 3D

Move elements along the Z-axis (towards/away from the viewer).

```elm
import Anim.Property.Translate as Translate
import Anim.Extra.View3D as View3D

-- View setup (add perspective to container)
view model =
    div [ id "container", View3D.perspective 1000 ]
        [ div ([ id "card" ] ++ CSS.transitionAttributes "card" model.animation)
            [ text "3D animated!" ]
        ]

-- Move element towards viewer
popOut builder =
    builder
        |> Translate.for "card"
        |> Translate.fromZ 0
        |> Translate.toZ 100
        |> Translate.build

-- Full 3D movement
flyIn builder =
    builder
        |> Translate.for "card"
        |> Translate.fromXYZ -200 -100 -500
        |> Translate.toXYZ 0 0 0
        |> Translate.build
```

**Remember**: Any time you animate the `Z` axis, add `View3D.perspective` to the container.

## Rotate in 3D

Rotate elements around any axis for card flips, barrel rolls, and more.

```elm
import Anim.Property.Rotate as Rotate

-- Flip a card horizontally (like turning a page)
flipCard builder =
    builder
        |> Rotate.for "card"
        |> Rotate.fromY 0
        |> Rotate.toY 180
        |> Rotate.build

-- Flip vertically (like a coin toss)
coinFlip builder =
    builder
        |> Rotate.for "coin"
        |> Rotate.fromX 0
        |> Rotate.toX 360
        |> Rotate.build

-- Barrel roll
barrelRoll builder =
    builder
        |> Rotate.for "plane"
        |> Rotate.fromXYZ 0 0 0
        |> Rotate.toXYZ 360 180 360
        |> Rotate.build
```

**Remember**: Any time you rotate on the `X` or `Y` axes, add `View3D.perspective` to the container.

## Scale in 3D

Scale elements along the Z-axis for depth effects.

```elm
import Anim.Property.Scale as Scale

-- Scale in depth
depthScale builder =
    builder
        |> Scale.for "box"
        |> Scale.fromZ 1
        |> Scale.toZ 2
        |> Scale.build

-- Full 3D scale
grow3D builder =
    builder
        |> Scale.for "box"
        |> Scale.fromXYZ 0.5 0.5 0.5
        |> Scale.toXYZ 1 1 1
        |> Scale.build
```

**Remember**: Any time you scale the `Z` axis, add `View3D.perspective` to the container.

## Combining 3D Transforms

Combine multiple 3D properties for complex effects:

```elm
cardFlipWithZoom : AnimBuilder -> AnimBuilder
cardFlipWithZoom builder =
    builder
        |> Rotate.for "card"
        |> Rotate.fromY 0
        |> Rotate.toY 180
        |> Rotate.build
        |> Scale.for "card"
        |> Scale.from 1
        |> Scale.to 1.2
        |> Scale.build
        |> Translate.for "card"
        |> Translate.fromZ 0
        |> Translate.toZ 50
        |> Translate.build
```

!!! warning "Backface visibility"
    When flipping elements 180°, use `View3D.backfaceVisibility View3D.Hidden` on the element to hide the back face.

## Performance Tips

!!! tip "GPU Acceleration"
    All 3D transforms are GPU-accelerated. They're as performant as 2D transforms.

!!! tip "Avoid layout thrashing"
    Keep perspective containers stable. Changing container dimensions during animation can cause performance issues.

!!! tip "Use transform-style"
    For nested 3D elements, use `View3D.transformStyle View3D.Preserve3D` to maintain 3D context.
