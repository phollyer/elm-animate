🎨 Smooth Move ElmUI Examples

Comprehensive demonstration of all 4 animation approaches using ElmUI instead of the core HTML package.

🚀 ElmUI vs HTML Comparison

ElmUI provides a more declarative and type-safe way to build UIs in Elm. This set of examples showcases how each of the 4 Smooth Move animation approaches can be implemented using ElmUI.

📋 Scroll
Scrolling animations using simple Cmd-based functions or choose more advanced Task-based scrolling with composable error handling. Perfect for document and container scrolling operations.

🔄 Sub
Subscription-based element positioning with frame-rate independent animations. Ideal for multiple simultaneous element animations.

⚡ CSS
Native browser CSS transitions for optimal performance. Hardware accelerated animations with excellent battery efficiency. Supports position, opacity, scale, rotation, and color animations.

📊 Choose Your Animation Approach

  • Scroll: Task-based scrolling with error handling - best for document and container scrolling
  • Sub: Subscription-based positioning with frame-rate independent updates - perfect for real-time element positioning
  • CSS: Native CSS transitions with hardware acceleration - best performance and battery efficiency
  • Ports: Web Animations API integration via JavaScript - advanced features and platform-specific optimizations
