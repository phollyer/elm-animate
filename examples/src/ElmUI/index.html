📊 Choose Your Animation Approach

  • Scroll: Task-based scrolling with error handling - best for document and container scrolling operations
  • Sub: Subscription-based positioning with frame-rate independent updates - perfect for real-time element positioning and property animations
  • CSS: Native CSS transitions with hardware acceleration - best performance and battery efficiency for smooth property animations
  • Ports: Web Animations API integration via JavaScript - advanced features and platform-specific optimizations
